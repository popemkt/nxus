# Other Integrations

Claude Code integrates with popular development environments.

## VS Code

The **Claude Code extension for VS Code** brings AI assistance directly into your editor:

- Install from the VS Code marketplace
- [Claude Code Extension](https://marketplace.visualstudio.com/items?itemName=anthropic.claude-code)

## JetBrains IDEs

The **Claude Code agent** works with JetBrains IDEs (IntelliJ, WebStorm, PyCharm, etc.):

- Available as a plugin for all JetBrains products
- [JetBrains Plugin Marketplace](https://plugins.jetbrains.com/)
