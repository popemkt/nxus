# Goose Setup & Usage Guide

Goose is an open-source, extensible AI agent that goes beyond code suggestions. It can install tools, execute commands, edit files, and test your code with any LLM.

## Features

- **Multi-LLM Support**: Works with Anthropic, OpenAI, Databricks, and more.
- **Extensible**: Use and build extensions to give Goose new capabilities.
- **Developer-First**: Designed to handle real-world developer workflows.

## Quick Start

### 1. Installation

If you haven't installed Goose yet, use the buttons below:

{{command:install}} (macOS/Linux)
{{command:install-windows}} (Windows)

### 2. Configuration

Set up your LLM providers:

{{command:configure}}

### 3. Start a Session

Once configured, start an interactive session:

{{command:session}}

Or try the web interface:

{{command:web}}

## Common Commands

- `goose session`: Start an interactive session.
- `goose run "do X"`: Run a one-off task.
- `goose web --open`: Open the web UI.
- `goose configure`: Manage providers and extensions.
- `goose update`: Keep Goose up to date.
